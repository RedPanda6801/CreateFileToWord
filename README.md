# CreateFileToWord
일정 양식에 맞춰진 워드 생성기

### 특징
- 필요에 의해 특정 파일을 샘플로 하는 워드 파일 생성기입니다.
- 추가하는 워드 파일은 반드시 개발자에게 문의하여 샘플 파일을 받으시기 바랍니다.
- 여러 학생 입력 시 줄 바꿈(또는 개행문자)로 구분하여 입력해주세요.
- 여러 파일 다운 시 크롬 기준 아래의 알림을 허용해주세요.

### 사용방법

1. URL 접속 : https://redpanda6801.github.io/CreateFileToWord/
2. 샘플 파일 추가 (sample.docx)
3. 일련번호 추가 (마지막 / 이후 숫자는 auto increment 됩니다. 반드시 정수 초기값을 입력하세요.)
4. 과정명 / 장소 / 날짜 입력 (텍스트이므로 형식 자유)
5. 학생명 입력 (여러명 가능, 줄바꿈으로 구분) 후 확인 버튼을 눌러 아래 리스트에 추가

6. 리스트 요소 중 왼쪽 빨간 x버튼을 누르면 리스트에서 삭제
7. 상장으로 출력 버튼 클릭 시 저장 파일 경로 (폴더) 입력
8. 다운로드 확인
